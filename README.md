# Node Api Test

This is a test project for a NodeJS position at VoIS.

## ⚙️ Installation

```bash
$ npm install
```

## 🏗️ Development

```bash
$ npm run start:dev
```

The server runs on `localhost:3000` and you can find the swagger docs on `localhost:3000/api`

## 🧪 Test

```bash
$ npm run test
```
